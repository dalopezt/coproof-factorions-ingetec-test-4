**Theorem.** The set of all natural numbers equal to the sum of the factorials of their own decimal digits is exactly $\{1, 2, 145, 40585\}$. Formally, if $\sigma(n)$ denotes the function sending a natural number $n$ to $\sum_{d \,\in\, \text{digits}(n)} d!$ (where $\text{digits}(n)$ is the list of decimal digits of $n$), then

$$\{ n \in \mathbb{N} \mid n = \sigma(n) \} = \{1,\, 2,\, 145,\, 40585\}.$$
